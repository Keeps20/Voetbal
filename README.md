# Project 0

Voor dit project is een website gemaakt over vrouwen 2 van VVLimmen.

homepage.html: Op deze pagine is een kort overzicht gegeven van wat er op de site te vinden is.

overwinningsfoto.html: Op deze pagina zijn verschillende foto's geplaatst, elk van een gewonnen wedstrijd.

uitslagen.html: Op deze pagina zijn twee tabellen weergegeven. In de ene tabel staan de wedstrijden die gespeeld zijn in de competitie
                en in de andere tabel zijn de gespeelde bekerwedstrijden weergegeven. In beide tabellen staan links de teams die tegen elkaar gespeeld hebben, in het midden staat de uitslag en rechts de datum waarop de wedstrijd gespeeld is.

spelers.html: Op deze pagina is een overzicht gegeven van alle spelers uit het team met hun bijbehorende positie.

In styles.css en extra.css staan verschillende CSS properties bij verschillende classes. Extra.css is gemaakt aan de hand van extra.scss. In deze file is gebruik gemaakt van een variabele, nesting en inheritance.
